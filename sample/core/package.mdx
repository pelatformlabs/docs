## Installing the package

To install the package, run the following command:

<CodeGroup>
  ```bash npm
  npm install @pelatform/<packages>
  ```

  ```bash yarn
  yarn add @pelatform/<packages>
  ```

  ```bash pnpm
  pnpm add @pelatform/<packages>
  ```

  ```bash bun
  bun add @pelatform/<packages>
  ```
</CodeGroup>
