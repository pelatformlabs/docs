# Get Started with Pelatform

Pelatform provides a comprehensive suite of development tools and UI components to help you build modern applications faster. Choose what you need based on your project requirements.

## Start Building in Three Steps

### Step 1: Choose Your Tools

Pelatform offers two main categories of tools:

<AccordionGroup>
  <Accordion icon="wrench" title="Development Toolkits">
    Essential development utilities and configurations for modern applications:

    <ul>
      <li><strong>Configs</strong>: ESLint, TypeScript, Biome configurations optimized for TypeScript projects</li>
      <li><strong>Utils</strong>: 100+ utility functions for validation, crypto, datetime, strings, URLs, and more</li>
      <li><strong>Email</strong>: Multi-provider email sending with React template support (Resend, SMTP)</li>
      <li><strong>Storage</strong>: Multi-provider file storage abstraction (S3, Cloudflare R2, Cloudinary, MinIO)</li>
    </ul>

    <Note>Perfect for backend APIs, full-stack applications, and development tooling.</Note>
  </Accordion>

  <Accordion icon="palette" title="UI Components">
    Complete UI component library for building modern, accessible interfaces:

    <ul>
      <li><strong>150+ Components</strong>: Comprehensive component library with consistent design</li>
      <li><strong>Modern Stack</strong>: Built with React, TailwindCSS, and Radix UI primitives</li>
      <li><strong>Accessible</strong>: WCAG compliant out of the box with keyboard navigation</li>
      <li><strong>TypeScript</strong>: Full type safety and IntelliSense support</li>
    </ul>

    <Note>Ideal for user interfaces, dashboards, and modern web applications.</Note>
  </Accordion>
</AccordionGroup>

### Step 2: Installation

<AccordionGroup>
  <Accordion icon="package" title="Install Development Toolkits">
    Install individual toolkits based on your needs:

    <PackageInstaller
      command="
    # Development configs
    installer @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

    # Utilities and helpers
    installer @pelatform/utils

    # Email and storage
    installer @pelatform/email @pelatform/storage
    " />

    <Tip>Choose only the packages you need to minimize bundle size.</Tip>
  </Accordion>

  <Accordion icon="box" title="Install UI Components">
    Install the complete UI library or individual packages:

    <PackageInstaller
      command="
    # Complete UI library
    installer pelatform-ui react react-dom

    # Or install specific packages
    installer @pelatform/ui.base @pelatform/ui.default @pelatform/ui.animation
    " />
    
    <Note>UI components require React 18+ and have peer dependencies like Radix UI and TailwindCSS.</Note>
  </Accordion>
</AccordionGroup>

### Step 3: Start Building

<AccordionGroup>
  <Accordion icon="code" title="Set Up Development Environment">
    Configure your development environment with Pelatform tools:

    ```js eslint.config.js
    import pelatform from '@pelatform/eslint-config';

    export default [
      ...pelatform,
      {
        rules: {
          // Your custom rules
        }
      }
    ];
    ```

    ```json tsconfig.json
    {
      "extends": "@pelatform/tsconfig/base.json",
      "compilerOptions": {
        "target": "ES2022",
        "module": "ESNext"
      }
    }
    ```
  </Accordion>

  <Accordion icon="sparkles" title="Create Your First Component">
    Start building with Pelatform UI components:

    ```typescript MyComponent.tsx
    import { Button, Card, Input, Form } from "pelatform-ui";

    function MyComponent() {
      return (
        <Card className="p-6">
          <h2 className="text-2xl font-bold mb-4">Welcome to Pelatform</h2>
          <Form className="space-y-4">
            <Input placeholder="Enter your email" type="email" />
            <Button className="w-full">Get Started</Button>
          </Form>
        </Card>
      );
    }

    export default MyComponent;
    ```
  </Accordion>
</AccordionGroup>
