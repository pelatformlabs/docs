# Get Started with Pelatform

Pelatform provides a comprehensive suite of development tools and UI components to help you build modern applications faster. Choose what you need based on your project requirements.

## Start Building in Three Steps

### Step 1: Choose Your Tools

Pelatform offers two main categories of tools:

<Cards>
  <Card title="Development Toolkits" icon="wrench">
    <ul>
      <li><strong>Configs</strong>: ESLint, TypeScript, Biome configurations</li>
      <li><strong>Utils</strong>: 100+ utility functions for common tasks</li>
      <li><strong>Email</strong>: SMTP and email template utilities</li>
      <li><strong>Storage</strong>: Multi-provider file storage abstraction</li>
    </ul>
  </Card>

  <Card title="UI Components" icon="palette">
    <ul>
      <li><strong>150+ Components</strong>: Complete UI component library</li>
      <li><strong>Modern Stack</strong>: React, TailwindCSS, Radix UI</li>
      <li><strong>Accessible</strong>: WCAG compliant out of the box</li>
      <li><strong>TypeScript</strong>: Full type safety and IntelliSense</li>
    </ul>
  </Card>
</Cards>

### Step 2: Installation

<AccordionGroup>
  <Accordion icon="package" title="Install Development Toolkits">
    Install individual toolkits based on your needs:

    <CodeGroup>
      ```bash npm
      # Development configs
      npm install @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

      # Utilities and helpers
      npm install @pelatform/utils

      # Email and storage
      npm install @pelatform/email @pelatform/storage
      ```

      ```bash yarn
      # Development configs
      yarn add @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

      # Utilities and helpers
      yarn add @pelatform/utils

      # Email and storage
      yarn add @pelatform/email @pelatform/storage
      ```

      ```bash pnpm
      # Development configs
      pnpm add @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

      # Utilities and helpers
      pnpm add @pelatform/utils

      # Email and storage
      pnpm add @pelatform/email @pelatform/storage
      ```

      ```bash bun
      # Development configs
      bun add @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

      # Utilities and helpers
      bun add @pelatform/utils

      # Email and storage
      bun add @pelatform/email @pelatform/storage
      ```
    </CodeGroup>

    <Tip>Choose only the packages you need to minimize bundle size.</Tip>
  </Accordion>

  <Accordion icon="box" title="Install UI Components">
    Install the complete UI library or individual packages:

    <CodeGroup>
      ```bash npm
      # Complete UI library
      npm install pelatform-ui react react-dom

      # Or install specific packages
      npm install @pelatform/ui.base @pelatform/ui.default @pelatform/ui.animation
      ```

      ```bash yarn
      # Complete UI library
      yarn add pelatform-ui react react-dom

      # Or install specific packages
      yarn add @pelatform/ui.base @pelatform/ui.default @pelatform/ui.animation
      ```

      ```bash pnpm
      # Complete UI library
      pnpm add pelatform-ui react react-dom

      # Or install specific packages
      pnpm add @pelatform/ui.base @pelatform/ui.default @pelatform/ui.animation
      ```

      ```bash bun
      # Complete UI library
      bun add pelatform-ui react react-dom

      # Or install specific packages
      bun add @pelatform/ui.base @pelatform/ui.default @pelatform/ui.animation
      ```
    </CodeGroup>

    <Note>UI components require React 18+ and have peer dependencies like Radix UI and TailwindCSS.</Note>
  </Accordion>
</AccordionGroup>

### Step 3: Start Building

<AccordionGroup>
  <Accordion icon="code" title="Set Up Development Environment">
    Configure your development environment with Pelatform tools:

    ```typescript
    // eslint.config.js
    import pelatform from '@pelatform/eslint-config';

    export default [
      ...pelatform,
      {
        rules: {
          // Your custom rules
        }
      }
    ];

    // tsconfig.json
    {
      "extends": "@pelatform/tsconfig/base.json",
      "compilerOptions": {
        "target": "ES2022",
        "module": "ESNext"
      }
    }
    ```
  </Accordion>

  <Accordion icon="sparkles" title="Create Your First Component">
    Start building with Pelatform UI components:

    ```typescript
    import { Button, Card, Input, Form } from "pelatform-ui";

    function MyComponent() {
      return (
        <Card className="p-6">
          <h2 className="text-2xl font-bold mb-4">Welcome to Pelatform</h2>
          <Form className="space-y-4">
            <Input placeholder="Enter your email" type="email" />
            <Button className="w-full">Get Started</Button>
          </Form>
        </Card>
      );
    }

    export default MyComponent;
    ```
  </Accordion>
</AccordionGroup>
