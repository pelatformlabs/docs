## Quick Start

<Steps>
  <Step title="Install Configurations">
    Install the configuration packages you need:

    <PackageInstaller command="
    # TypeScript configuration
    installer -D @pelatform/tsconfig typescript

    # Biome configuration
    installer -D @pelatform/biome-config @biomejs/biome

    # ESLint configuration
    installer -D @pelatform/eslint-config eslint
    " />
  </Step>

  <Step title="Configure TypeScript">
    Create your `tsconfig.json`:

    ```json tsconfig.json
    {
      "$schema": "https://json.schemastore.org/tsconfig.json",
      "extends": "@pelatform/tsconfig/base"
    }
    ```
  </Step>

  <Step title="Configure Biome">
    Create your `biome.jsonc`:

    ```jsonc biome.jsonc
    {
      "$schema": "./node_modules/@biomejs/biome/configuration_schema.json",
      "extends": ["@pelatform/biome-config/base"]
    }
    ```
  </Step>

  <Step title="Add Scripts">
    Add build scripts to your `package.json`:

    ```json package.json
    {
      "scripts": {
        "build": "tsc",
        "type-check": "tsc --noEmit",
        "format": "biome format --write",
        "lint": "biome lint",
        "lint:fix": "biome lint --write"
      }
    }
    ```
  </Step>

  <Step title="Run Checks">
    Execute common tasks:

    <PackageRun command="run type-check" />
    <PackageRun command="run lint" />
    <PackageRun command="run lint:fix" />
    <PackageRun command="run format" />
  </Step>
</Steps>

## Usage Examples

### React Application

```json tsconfig.json
{
  "extends": "@pelatform/tsconfig/react",
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"],
      "@/components/*": ["./src/components/*"]
    }
  }
}
```

  ```jsonc biome.jsonc
  {
    "extends": ["@pelatform/biome-config/base"],
    "files": {
      "ignore": [".next/**/*", "out/**/*"]
    }
  }
  ```

### Node.js API

```json tsconfig.json
{
  "extends": "@pelatform/tsconfig/node",
  "compilerOptions": {
    "rootDir": "src",
    "outDir": "dist"
  }
}
```

```jsonc biome.jsonc
{
  "extends": ["@pelatform/biome-config/base"],
  "javascript": {
    "globals": ["Buffer", "process", "global"]
  }
}
```

### Library Package

```json tsconfig.json
{
  "extends": "@pelatform/tsconfig/base",
  "compilerOptions": {
    "declaration": true,
    "declarationMap": true,
    "moduleResolution": "bundler"
  }
}
```

## Integration with Development Tools

### VS Code Setup

```json .vscode/settings.json
{
  "editor.defaultFormatter": "biomejs.biome",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.biome": "explicit"
  },
  "typescript.preferences.importModuleSpecifier": "relative"
}
```

### GitHub Actions

```yaml .github/workflows/ci.yml
name: CI
on: [push, pull_request]
jobs:
  check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci
      - run: npm run type-check
      - run: biome check
```
