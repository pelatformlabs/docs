## Quick Start

<Steps>
  <Step title="String Utilities">
    Manipulate and format strings:

    ```typescript
    import { cn, truncate, capitalize, camelCase, slugify } from '@pelatform/utils';

    // Tailwind class merging
    const className = cn('px-4 py-2', 'bg-blue-500', { 'opacity-50': disabled });

    // Text truncation
    const short = truncate('This is a very long text', { length: 20 });
    // "This is a very l..."

    // Capitalize
    const title = capitalize('hello world');
    // "Hello World"

    // Camel case
    const camel = camelCase('hello world example');
    // "helloWorldExample"

    // Slugify
    const slug = slugify('Hello World! This is Awesome');
    // "hello-world-this-is-awesome"
    ```
  </Step>

  <Step title="Datetime Utilities">
    Format and manipulate dates:

    ```typescript
    import {
      formatDateTime,
      formatDateTimeSmart,
      timeAgo,
      getDaysDifference,
      getFirstAndLastDay
    } from '@pelatform/utils';

    const date = new Date('2024-01-15T14:30:00Z');

    // Smart formatting
    const smart = formatDateTimeSmart(date);
    // "January 15, 2024 at 2:30 PM" (based on current date)

    // Custom formatting
    const formatted = formatDateTime(date, 'MMM dd, yyyy');
    // "Jan 15, 2024"

    // Relative time
    const relative = timeAgo(date);
    // "2 months ago" (or actual time difference)

    // Days between dates
    const days = getDaysDifference(date, new Date());
    // 45

    // Month boundaries
    const { firstDay, lastDay } = getFirstAndLastDay('2024-01');
    // First: 2024-01-01, Last: 2024-01-31
    ```
  </Step>

  <Step title="Crypto Utilities">
    Generate IDs and handle security:

    ```typescript
    import { nanoid, cuid, hashStringSHA256 } from '@pelatform/utils';

    // Generate unique IDs
    const nanoId = nanoid();
    // "V1StGXR8_Z5jdHi6B-myT"

    const cuidId = cuid();
    // "ck4b8v9pg0001mjl1m7g3h123"

    // Hash strings
    const hash = await hashStringSHA256('my-secret-data');
    // "a591a6d40bf420404a011733cfb7b190d62c65bf0bcda32b57b277d9ad9f146e"
    ```
  </Step>

  <Step title="Validation Utilities">
    Validate data and deep compare objects:

    ```typescript
    import {
      validateEmail,
      isValidUrl,
      deepEqual,
      isIframeable
    } from '@pelatform/utils';

    // Email validation
    const emailResult = validateEmail('user@example.com');
    // { isValid: true, normalized: 'user@example.com' }

    // URL validation
    const urlValid = isValidUrl('https://example.com');
    // true

    // Deep equality check
    const obj1 = { a: 1, b: { c: 2 } };
    const obj2 = { a: 1, b: { c: 2 } };
    const isEqual = deepEqual(obj1, obj2);
    // true

    // Check if URL can be iframed
    const canIframe = isIframeable('https://example.com');
    // false (X-Frame-Options)
    ```
  </Step>
</Steps>

## Server-Only Utilities

Server utilities provide essential security and authentication functions:

- **Password Security**: BCrypt hashing, verification, secure password generation
- **JWT Operations**: Token generation, verification, expiration handling
- **Cryptographic Functions**: Secure random string generation
- **HTTP Security**: IP extraction, request parsing, user agent detection
- **Advanced DateTime**: Natural language parsing for server environments

```typescript
import {
  hashPassword,
  verifyPassword,
  generateJWT,
  verifyJWT,
  generateRandomString
} from '@pelatform/utils/server';

// Password hashing (bcrypt)
const hashed = await hashPassword('password123');
const isValid = await verifyPassword('password123', hashed);

// JWT with expiration
const token = generateJWT(
  { userId: '123', role: 'admin' },
  'your-secret',
  { expiresIn: '7d' }
);
```

## Best Practices

<Accordion title="Tree Shaking">
  Import only what you need for optimal bundle size:

  ```typescript
  // ✅ Good: Tree shakable
  import { cn, truncate } from '@pelatform/utils';

  // ❌ Avoid: Import entire bundle
  import * as utils from '@pelatform/utils';
  ```
</Accordion>

<Accordion title="Error Handling">
  Many utils are safe by design, but handle errors appropriately:

  ```typescript
  try {
    const result = validateEmail('user@example.com');
    if (!result.isValid) {
      return { error: result.error };
    }
  } catch (error) {
    logger.error('Email validation failed', error);
    return { error: 'Validation failed' };
  }
  ```
</Accordion>

<Accordion title="Performance">
  Use performance utilities to optimize your app:

  ```typescript
  // Debounce search inputs
  const debouncedSearch = debounce(handleSearch, 300);

  // Throttle resize handlers
  const throttledResize = throttle(handleResize, 100);

  // Use nanoid for unique IDs instead of timestamps
  const id = nanoid();
  ```
</Accordion>

<Accordion title="TypeScript">
  Leverage full type safety:

  ```typescript
  import type {
    FormatDateTimeOptions,
    CurrencyFormatterOptions,
    ValidationResult
  } from '@pelatform/utils';

  const format = (date: Date, options: FormatDateTimeOptions) => {
    return formatDateTime(date, options.format);
  };
  ```
</Accordion>
