## Quick Start

Get started with Pelatform Toolkits in minutes. Install individual packages based on your needs:

<Steps>
  <Step title="Install Configs">
    Set up your development environment with our configurations:

    <PackageInstaller command="
    # Install all configurations
    installer @pelatform/eslint-config @pelatform/tsconfig @pelatform/biome-config

    # Install with Vite support
    installer @pelatform/eslint-config-vite
    " />
  </Step>

  <Step title="Add Utilities">
    Include our utility library for common functions:

    <PackageInstaller command="installer @pelatform/utils" />
  </Step>

  <Step title="Setup Communication">
    Add email capabilities to your application:

    <PackageInstaller command="installer @pelatform/email" />
  </Step>

  <Step title="Configure Storage">
    Add multi-cloud storage support:

    <PackageInstaller command="installer @pelatform/storage" />
  </Step>
</Steps>

## Architecture Overview

Pelatform Toolkits are designed to work together seamlessly while remaining modular and independently usable:

### Configuration Foundation
Start with our configuration packages to establish code quality and development standards across your team.

### Utility Layer
Leverage our comprehensive utility library for common operations, validation, and helper functions.

### Service Integration
Add email and storage capabilities with consistent APIs that work across multiple providers.

### Production Ready
All tools are battle-tested in production environments with comprehensive error handling and monitoring.
