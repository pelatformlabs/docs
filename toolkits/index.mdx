Pelatform Toolkits is a comprehensive collection of development tools designed to accelerate modern SaaS application development. Our toolkits provide everything you need from code quality enforcement to production-ready communication and storage solutions.

## Available Toolkits

Explore our specialized packages designed for modern development:

<Columns cols={1}>
  <Card title="Configurations" icon="settings" href="/toolkits/config">
    Standardized configurations for Biome, ESLint, and TypeScript to ensure code quality and consistency across your projects.
  </Card>

  <Card title="Email Service" icon="mail" href="/toolkits/email">
    Multi-provider email solution supporting Resend and SMTP with React-based templates and automatic provider detection.
  </Card>

  <Card title="Storage Solution" icon="cloud" href="/toolkits/storage">
    Unified storage interface for AWS S3, Cloudflare R2, Cloudinary, and compatible providers with file management helpers.

  <Card title="Utility Library" icon="code" href="/toolkits/utils">

  </Card>
</Columns>

## Quick Start

Get started with Pelatform Toolkits in minutes. You can install individual packages or the complete suite.

<Steps>
  <Step title="Install Configs">
    Set up your development environment with our configurations:
    
    <PackageInstaller command="installer -D @pelatform/biome-config @pelatform/eslint-config @pelatform/tsconfig" />
  </Step>

  <Step title="Add Core Libraries">
    Install the utility and service packages:

    <PackageInstaller command="installer @pelatform/email @pelatform/storage @pelatform/utils" />
  </Step>
</Steps>

## Common Use Cases

### New SaaS Project
Initialize a new project with the complete toolkit stack for a solid foundation.

<PackageInstaller command="installer @pelatform/utils @pelatform/email @pelatform/storage -D @pelatform/eslint-config @pelatform/tsconfig" />

### React/Next.js Application
Use our optimized ESLint config and utilities for frontend development.

<PackageInstaller command="installer @pelatform/utils -D @pelatform/eslint-config-react" />

### Backend Service
Leverage server-side utilities and storage for API services.

<PackageInstaller command="installer @pelatform/utils @pelatform/storage @pelatform/email" />
